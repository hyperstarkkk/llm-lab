# llm-lab
This is a roadmap for newbees who wants to build a comprehensive and solid foundation for large language model. This roadmap does will prioritize well-known tutorials, and only invent wheels when they are not exist.

# Roadmap
This roadmap contains the materials of the following topics:
1. Understanding Transformer & Train your GPT2: [LLMs-from-scratch, Sebastian Raschka](https://github.com/rasbt/LLMs-from-scratch)
2. Implement a nano-vllm: [nano-vllm, GeeeekExplorer(Xingkai Yu)](https://github.com/GeeeekExplorer/nano-vllm)
3. Placeholder
